One line wonders

This, in my opinion, is a pretty useful website. You can generate birthday, valentine, and other types of wonderful cards and messages, all in one line of JavaScript!
To run these snippets, you have to open the console. You can do this by right-clicking the page, and selecting the inspect (sometimes inspect element) option. Then, go to the console tab, and paste in the code you want to be run!
