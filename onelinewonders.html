A basic b-day card:

A classic b-day card! You can't go wrong with this, so give it a try! Remember - type \n to get a line break, but dont type any other back-slashes, as that could break the code! Because of the f-strings, this only works in python 3.6=<

print(f"\033[35m{'Happy B-day {your name here}!'.capitalize()}\033[90m\n{chr(60)+(l_1 := '------~~~~~~**') + l_1[::-1]+chr(62)}\033[0m\n\n{'{your message here}'}\n\033[90m{chr(60)+l_1+l_1[::-1]+chr(62)}\033[0m")
