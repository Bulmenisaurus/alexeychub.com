Explanation:

What is CSS?

How everything looks like in the web, whether you want it or not, is controlled by CSS. CSS lets you do pretty awesome stuff, like change the color of text, make borders, or even rotate content!
For example, the following CSS: 

p {
 color: red;
 font-family: Arial, Helvetica, sans-serif;
}

Would tell the browser to set the color of all paragraphs (p for short) to red, and make the font of all paragraphs Arial. In reality, there are over 500 css properties, and there are elements other than <p> present on a page.

What do these weird symbols mean?

Sometimes you might see some strange symbols appear near the sliders.

Id:

The hash (#) represents the ID of one singular element on the page. For example, #link {color: red} means that a single element on the page with the id of link will become red.

Class:

A dot (.) represents a CLASS of one or multiple elements on a page. For example, .text-container {font-style: italic} will make every single element with the class of text-container be italic.
