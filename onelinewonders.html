A basic birthday card:

A classic b-day card! You can't go wrong with this, so give it a try! Remember - \n will create a new line, but dont type any other back-slashes, as that could break the code! 

print(f"{'Happy B-day {your name here}!'.capitalize()}\n{chr(60)+(l_1 := '------~~~~~~**') + l_1[::-1]+chr(62)}\n\n{'{your message here}'}\n\n{chr(60)+l_1+l_1[::-1]+chr(62)}")
