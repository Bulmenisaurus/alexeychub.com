XMLScript

A new, XML-based programming language. Embrace the simplicity of python while leveraging xml's inuitive and built-in string formatting.

Simple and easy-to-learn syntax

You can learn XMLS in under an hour!

<?xml version="1.0" encoding="UTF-8"?>
<code>
    <print>Hello, <b>XMLScript</b>!</print>

    <!-- or, you can create a function -->

    <function name="greet" who="">
        <print>Hello, <who />!</print>
    </function>

    <greet who="functions" />
</code>
Hello, XMLScript!
Hello, functions!
